【Markdown】基础用法

发表于2022-09-21|更新于2023-02-23|博客
|阅读量:

简介

Markdown是一门比html更简单的标记语言，主要用于日常写作，再通过相应的编辑器或脚本转换成html用于页面渲染。Markdown的主要特点就是简单，语法也简单易上手

应用

  • 用来写博客、邮件
  • Markdown 很容易转化成 HTML 或者 PDF
  • git 的 README.md 就是 Markdown 格式

工具

我使用的是VS Code(Visual Studio Code)
VS Code下载
安装Markdown Preview Enhanced插件就能实现在线预览效果，右键游览页面也可以实现浏览器预览，转换为html、PDF等功能。

也可以使用 Byword、Ulysses、Typora、Markdown Pad 2等工具

基础语法

标题

Markdown支持6种级别的标题，使用#在行首，对应html标签 h1 ~ h6

# h1
## h2
### h3
#### h4
##### h5
###### h6

另一种是使用===或者---表示（不推荐）

分割线

使用三个或以上的 - 或者 * 表示，且这一行只有符号，注意不要被识别为二级标题即可，例如中间或者前面可以加空格。

    段落及区块引用

    使用 > 表示，可以有多个 >，表示层级更深，例如
    > 段落一
> > 段落二
> > > 段落三

    插入代码块

    行内代码块用单个反引号 ` 包裹即可
    插入一整段代码，需要至少使用两个以上反引号进行包裹。

    表格

    表头|条目一|条目二

项目|项目一|项目二

三个短斜杠左右的冒号用于控制对齐方式，只放置左边冒号表示文字居左，只放置右边冒号表示文字居右，如果两边都放置冒号表示文字居中。

特殊符号处理

Markdown使用反斜杠\插入语法中用到的特殊符号。

\   反斜线
`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧
#   井字号
+   加号
-   减号
.   英文句点
!   惊叹号

标签和分类

一般在文首输入tags添加标签，categories添加分类。分类具有层次性，标签无层次性。

tags: 
    - Markdown
    - 语言
categories: 
    - 技术
    - 博客

hexo不支持一篇文章设置多个同级分类，支持父子分类。例如我示例中的分类，会使博客成为技术的子分类。
