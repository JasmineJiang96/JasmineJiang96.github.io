Linux背景介绍

简介

Linux有两种含义：一种是Linus编写的开源操作系统的内核，另一种是广义的操作系统

执行环境：云主机；无数据的PC（不推荐多系统混跑）；虚拟机（推荐方式）

常用版本

内核版本

  • https://www.kernel.org/
  • 内核版本分为三个部分
  • 主版本号、次版本号、末版本号
  • 次版本号是奇数开发版，偶数为稳定版

发行版本

RedHat Enterprise Linux 收费，软件经过了专业人员测试，非常稳定

Fedora 免费，没有专业人员测试，稳定性差

CentOS 免费且稳定

Debian、Ubuntu 华丽界面

虚拟机和系统安装

安装

虚拟机工具VirtualBox下载： https://www.virtualbox.org/wiki/Downloads

镜像下载：http://mirrors.aliyun.com/centos/7/isos/x86_64/ 选择centOS7或者centOS8，DVD的版本

下载完成后在VirtualBox里安装系统镜像文件即可

终端

  • 图形终端
  • 命令行终端
  • 远程终端（SSH、VNC）

常见目录介绍

  • / 根目录
  • /root root用户的家目录
  • /home/username 普通用户的家目录
  • /etc 配置文件目录
  • /bin 命令目录
  • /sbin 管理命令目录
  • /usr/bin /usr/sbin 系统预装的其他命令

关机命令

Linux作为服务器系统，一般不会关机，练习时可以用命令关机 init 0

帮助命令

  • man帮助

是manual的缩写；

man 命令名 例如man ls 就是查看ls命令和参数的用法，按q键退出

man也是一条命令，分为9章，可以使用man命令获取man的帮助，例如man 7 man

man -a 关键字 查看关键字的帮助，第一个章节是命令，不想看按q键退出，然后回车可以看下一章节，ctrl+D跳过，ctrl+C退出

  • help帮助

shell（命令解释器）自带的命令成为内部命令，其他的是外部命令；type命令可以区分命令类型，例如type cd

内部命令使用help帮助，例如help cd

外部命令使用help帮助，例如ls --help

  • info帮助

比help更详细，作为help的补充，例如，info ls，但是是英文版的

  • 使用网络资源（搜索引擎和官方文档）

系统操作

文件管理

文件查看

  • pwd 显示当前的目录名称
  • cd 更改当前的操作目录

cd /path/to/…… 绝对路径

cd ./path/to/…… 相对路径

cd ../path/to/……相对路径

  • ls 查看当前目录下的文件

ls [选项] [文件名……]

显示颜色不同代表文件夹权限不同

常见参数：

-l 长格式显示文件

-a 显示隐藏文件

-r 逆序显示

-t 按照时间顺序显示

-R 递归显示

文件操作

目录文件的创建与删除
  • mkdir 建立目录

常用参数 -p 建立多级目录

mkdir /a 根目录下建立一个目录a

mkdir ./a 或者 mkdir a 当前目录下建立一个目录a

mkdir b c d 当前目录下建立多个目录b c d

创建重名目录会创建失败

mkdir -p a/b/c/d/e/f/g 创建多级目录

  • rmdir 删除空目录
  • rm 删除文件

常用参数 -r 删除目录（包括目录下的所有文件） -f 删除文件不进行提示

rm -r /a 删除目录a，需要确认子级目录是否删除

rm -rf 删除目录，不进行提示是否删除目录，谨慎使用，例如rm -rf / a 中间加个空格会导致根目录删除且没有提示

复制和移动目录
  • cp 复制文件和目录

不加参数-r时只能复制文件，cp -r 复制目录

cp -r /root/a /tmp 将root下a目录复制到tmp下

cp /filea /tmp 将根目录下的文件filea复制到tmp下

cp -p 复制时保留用户、权限、时间等文件属性

cp -a 等同于-dpR

  • mv 文件和文件夹的移动、重命名

mv /filea /fileb 将根目录下的文件filea重命名为fileb

mv /fileb /tmp 将根目录下的文件fileb移动到/tmp

mv /tmp/fileb /filec 将/tmp目录下的文件fileb移动到根目录并重命名为filec

通配符

定义：shell 内建的符号

用途：操作多个相似（有简单规律）的文件

常用通配符：

  • * 匹配任何字符串

cp file* / 复制当前目录下file开头后面无论多少个字符的文件到根目录下

  • ? 匹配一个字符串

cp file? / 复制当前目录下file开头后面一个字符的文件到根目录下

  • [xyz] 匹配xyz任意一个字符
  • [a-z] 匹配一个范围
  • [!xyz] 或 [^xyz] 不匹配

文本内容查看

  • cat 文本内容显示到终端

  • head 查看文件开头

    默认显示文件开头10行

    head -5 /filea 查看根目录下的文件filea的前5行

  • tail 查看文件结尾

    • 常用参数 -f 文件内容更新后，显示信息同步更新，ctrl+c停止

  • wc 统计文件内容信息

    wc -l /filea 查看根目录下的文件filea有多少行

打包压缩和解压缩

打包

Linux的备份压缩

  • 最早的Linux备份介质是磁带，使用的打包命令是tar
    • 常用参数 c 打包 、x 解包、 f 指定操作类型为文件、 -C 指定需要解压到的目录

tar cf /tmp/etc-backup.tar /etc 把根目录下的ect文件夹打包成tmp下的etc-backup.tar文件

tar xf /tmp/etc-backup.tar -C /root 把/tmp下的etc-backup.tar文件解压到/root下

压缩与解压缩
  • 可以打包后的磁带文件进行压缩储存，压缩的命令是gzip和bzip2
    • gzip压缩速度更快，bzip2压缩比例更高

tar czf /tmp/etc-backup.tar.gz /etc （解压参数是xzf）等同于 gzip /tmp/etc-backup.tar.gz /etc

tar cjf /tmp/etc-backup.tar.bz2 /etc （解压参数是xjf）等同于 bzip2 /tmp/etc-backup.tar.bz2/etc

  • 经常使用的扩展名是 .tar.gz .tar.bz2 .tgz .tbz2
    • .tgz是.tar.gz的缩写
    • .tbz2是.tar.bz2的缩写
