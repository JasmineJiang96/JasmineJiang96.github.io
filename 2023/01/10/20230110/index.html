Angular入门篇6 —— 表单

发表于2023-01-10|更新于2024-01-19|前端
|阅读量:

简介

Angular 提供了两种不同的方法来通过表单处理用户输入：响应式表单和模板驱动表单。

模板驱动表单

表单的控制逻辑写在组件模板中，适合简单的表单类型

创建

  • 引入依赖模块FormModules;
    ```typeScript
    import { FormsModule } from “@angular/forms”

@NgModule({
imports: [FormsModule],
})
export class AppModule {}

- DOM表单转换为ngForm
在form标签上添加 #form1=“ngForm”，创建NgForm实例
```html
<form #form1="ngForm" (ngSubmit)="onSubmit(form1)"></form>
  • 声明表单元素为ngModel
    ngModel会创建FormControl对象，绑定的值为model.username
    <form #form1="ngForm" (submit)="onSubmit(f)">
  <!-- 使用[(ngModel)]时，必须为元素定义一个name属性，Angular会相当于将formController绑定到FormGroup上 -->
<input type="text" [(ngModel)]="formData.username" name="username"/>
<button type="submit">提交</button>
</form>

    表单验证

  • required 必填字段
  • minlength 字段最小长度
  • maxlength 字段最大长度
  • pattern 验证正则 例如：pattern=”\d” 匹配一个数值
    <form #form1="ngForm" (submit)="onSubmit(f)">
  <!-- 使用[(ngModel)]时，必须为元素定义一个name属性，Angular会相当于将formController绑定到FormGroup上 -->
  <input type="text" [(ngModel)]="formData.username" name="username" required/>
  <!-- 验证未通过时的提示信息 -->
  <div *ngIf="username.touched && !username.valid && username.errors">
  <div *ngIf="username.errors.required">请填写用户名</div>
  <div *ngIf="username.errors.pattern">不符合正则规则</div>
</div>
  <!-- 表单整体未通过验证时禁用提交表单 -->
  <button type="submit" [disabled]="f.invalid">提交</button>
</form>

import { Component} from '@angular/core';
import { NgForm} from '@angular/forms';

@Component({
  selector: 'reactive-form',
  templateUrl: './reactive-form.component.html'
})
export class ReactiveFormComponent{
    formData = new User('');
    constructor(){};

    onSubmit(f: NgForm) {
        console.log('form value', f);
        // 通过f.value获取表单的值
        console.log('submit value', f.value);
        // 通过f.value获取表单是否校验通过
        console.log('submit valid', f.valid);
     }
}

// 1. 定义类，将属性当作表单的控件
class User {
  constructor(
    public username: string
  ) {}
}
